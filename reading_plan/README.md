## Usage

1. 引数にテクストのタイトル，ページ数，期日を渡し，関数を実行．
例えば，
```py3:
reading_plan("『消費社会の神話と構造』", 310, (2017,10,31))
```
2. Current page?: と聞かれるので答える．
